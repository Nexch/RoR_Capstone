## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

Ruby
Rails
Postgres

### Setup

$ git clone the repo
   Install gems with:
      bundle install
  Setup database with:
      rails db:create
      rails db:migrate

### Usage

Start server with:

```
    rails server
```

Open `http://localhost:3000/` in your browser.

### Run tests

```
    rpsec 
```
